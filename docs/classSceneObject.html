Detailed Description

Base class that scene objects inherit from. A scene object is a general purpose object. It's similar to the "entity" in an Entity-component-system (ECS) pattern. It's an object composed of multiple components such as rendering and physics components. In contrast to a real ECS, there is no memory optimization, e.g. similar components are not stored next to each other for cache-friendliness. Neither are there any systems, instead the data and behaviour is contained inside the components.

A scene object must be part of a scene.
