Detailed Description

A wheel with a built-in DC-motor. The DC-motor is modeled with basic DC-motor equations computing torque/force from a voltage input. Tune voltageInConstant and angularSpeedConstant for different acceleration and top speed (see tools/dc_motor_plot.py)
