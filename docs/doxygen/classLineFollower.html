Detailed Description

LineFollower - a robot that attempts to follow a path laid out as a black (or sometimes white) stripe.

The class provides pre-defined LineFollower specs as Blueprints. The class is flexible, and you can create a new Blueprint/Specification if you want to better resemble your own LineFollower. See BaseBot for a list of specification parameters.
